# DAEKWANG TECH Homepage V2 Compatible

이미지 호환성을 위해 WebP 대신 JPG 에셋을 사용하는 버전입니다.

## 사용법
1. ZIP 압축을 풉니다.
2. `index.html`을 더블클릭합니다.
3. 배포할 때는 이 폴더 전체를 업로드합니다. `assets` 폴더를 빼면 이미지가 깨집니다.

## 포함 페이지
- index.html
- technology.html
- products.html
- process.html
- quality.html
- facility.html
- company.html
